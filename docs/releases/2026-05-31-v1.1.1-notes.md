# Projectionist v1.1.1

## Highlights

- Fixes the Stats endpoint, the /Users list, and HiddenLibrary auto-hide on Jellyfin 10.11.9+ where `IUserManager.Users` was removed from the interface.

## Background

Jellyfin 10.11.9 dropped the `Users` property from `IUserManager`. Plugins compiled against 10.11.0-10.11.8 referenced it directly and threw `MissingMethodException` at runtime on a 10.11.9+ host. The replacement is the `GetUsers()` method (which returns a `User[]`).

This release adds a small ABI-compatibility shim that prefers `GetUsers()` and falls back to the old `Users` property for older Jellyfin hosts, so the same DLL works across 10.11.0 → 10.11.10.

## Upgrade Notes

- Restart Jellyfin after installing the plugin update.
- No configuration changes required.
