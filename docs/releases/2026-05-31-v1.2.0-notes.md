# Projectionist v1.2.0

## Highlights

### New features

- **Genre-aware preroll selection.** Library rules accept a list of genres so the same library can use different prerolls per genre.
- **Per-feature opt-out.** Mark specific items as "never preroll this".
- **Skip-rate analytics.** Tracks at what second users skip each intro and surfaces avg + skip-rate per file.
- **Outro / post-roll.** Drop videos into a post-roll folder; they play AFTER the feature ends.
- **Cinema countdown overlay.** Optional 5-4-3-2-1 countdown with feature title rendered over the preroll.
- **Coming-soon trailer.** Prepend a randomly-picked trailer from an unwatched movie.
- **Audio loudness analysis (MVP).** ffmpeg volumedetect flags prerolls that drift far from baseline.

### Fixed

- Skip button now actually appears during preroll playback.
- Maturity gate supports European systems (FSK, CSA, ACB, Eirin, ICAA, Medierådet).
- Schedule rules validate inputs and log warnings instead of silently never matching.
- Cooldown store no longer grows unbounded — auto-prunes old entries.
- Discovery skips hidden directories.
- Thread-safe RNG via Random.Shared.
- HiddenLibraryManager.FindItem is path-only — no ambiguous filename fallback.
- HideFromAllUsers serialized with SemaphoreSlim.

### Polish

- MaturityRanker lifted to its own public class for testability.
- MD5 uses MD5.HashData (less allocation, identical output).
- ~30 unit tests across MaturityRanker, ScheduleRule, CooldownStore, PrerollSelector.
- New CI workflow runs dotnet build -warnaserror + dotnet test on PRs.
- README updated with badges, FAQ, FileTransformation explainer, fixed install link + TOC.
- LICENSE + CHANGELOG.md bundled in release zip.
- .editorconfig + Dependabot + SECURITY.md + CONTRIBUTING.md + issue templates + PR template.

## Upgrade Notes

- Restart Jellyfin after install.
- All new features are opt-in via the admin UI.
- No data migration required.
