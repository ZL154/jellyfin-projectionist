# Projectionist v1.1.0

## Highlights

- Adds separate session behaviour controls for movies and TV episodes.
- Adds Feature Preload modes: Off, Warm, and experimental Hot.
- Adds Projectionist to the Jellyfin dashboard sidebar.
- Improves episode preroll handling across more Jellyfin Web playback paths.
- Fixes admin page controls so checkboxes, inputs, and dropdowns stay usable.

## Upgrade Notes

- Restart Jellyfin after installing the plugin update.
- Hard-refresh Jellyfin Web after upgrading so the updated playback hook is loaded.
- Hot preload is experimental and web-client only; it may briefly create an extra stream or transcode request while the preroll plays.

## Package

- Jellyfin ABI: 10.11.0.0
- Asset: `projectionist_1.1.0.0.zip`
